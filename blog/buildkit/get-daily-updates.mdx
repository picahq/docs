## Steps

1. **Connect Linear and Google Sheets**  
   Set up API keys for Linear and Google Sheets.

2. **Configure the Agent Tool**  
   Use the following prompt to instruct your agent:
   ```markdown
Create a Vercel AI SDK tool getDailyUpdates that, via Pica discovery (no hardcoding), fetches today’s Linear issues and appends them to Google Sheets. Inputs: spreadsheet URL/ID, sheet (default Sheet1), timezone (default UTC), maxIssues (default 100). Build from action knowledge (Linear “Query Issues”, Sheets “Append Values”), and append rows ["Date","Assignee","Status","Issue","Title","URL"] using /spreadsheets/{id}/values/{sheet}:append with valueInputOption=USER_ENTERED.   ```

3. **Run the Agent**  
   Trigger the tool and check your Google Sheet for today’s tickets, grouped by assignee and status.

## Environment Variables

- `LINEAR_API_KEY`
- `GOOGLE_SHEETS_API_KEY`
- `GOOGLE_SHEET_ID`

## Example Output

Your Google Sheet will show today’s Linear tickets, grouped by assignee, with separate sections for In Progress and Closed tickets.
