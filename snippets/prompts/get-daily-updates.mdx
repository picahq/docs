### Prompt the LLM to build your tool

 <Tab title="Get Daily Updates">
    <Card>
      Copy this prompt:
      ```markdown Agent Prompt expandable
      Create me a tool called getDailyUpdates in my agent for fetching today’s tickets from Linear, saving them into Google Sheets in a well-structured, easy-to-read format, grouped by assignee and status (In Progress / Closed).
      ```

      ## Environment Variables
      - `LINEAR_API_KEY`
      - `GOOGLE_SHEETS_API_KEY`
      - `GOOGLE_SHEET_ID`
      
      Note: The Google Sheet should clearly show today’s Linear tickets, grouped by assignee and status.
